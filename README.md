# Compendium
Website für relevanten Lehrstoff und Checklisten. Relevante Inhalte ab der 9. Klasse.
## Welche Bücher?
Der Inhalt dieser Seite wird aus dem folgenden Buch übernommen:
- **Französisch:** À plus! - Band 3 - Cornelsen

Wenn du dieses Buch in der Schule nutzt, kannst du diese Seite als kompaktes Mini-Buch nutzen!

## Das kleingedruckte
Bitte nutze diese Seite nicht als Spickzettel bei Tests. Vor allem solltest du dein Handy nicht in die Hülle deines Taschenrechners packen, damit es so aussieht, wie ein unschuldiger Taschenrechner. Und komm gar nicht auf die Idee, ein zweites Handy mitzubringen, falls das andere vor dem Unterricht eingesammelt wird.
